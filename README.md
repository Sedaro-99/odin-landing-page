This is recreation of landing page from Odin Project.

## Goal of the project
The main goal of the project was to verify the understanding of the flexbox concept and work without hand-holding. Only the .png visual and list of fonts/colors were given.

## Cíl projektu
Cílem projektu bylo utvrdit si znalosti flexboxu a začít pracovat samostatně. Podkladem pro landing page byl čistě obrázek a seznam fontů/barev, ale žádné další pomocné instrukce.
