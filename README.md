This is recreation of landing page from The Odin Project.

## Goal of the project
The main goal of the project was to verify the understanding of the flexbox concept and work without hand-holding. Only the .png visual and list of fonts/colors were given. The website is not supposed to be responsive.

## Cíl projektu
Cílem projektu bylo utvrdit si znalosti flexboxu a začít pracovat samostatně. Podkladem pro landing page byl čistě obrázek a seznam fontů/barev, ale žádné další pomocné instrukce. Web nemá být responzivní.
